使用说明 User Guide

点检流程说明

  1. 填写设备名称和生产线体信息
  2. 按顺序检查每个项目并选择状态
  3. 对于异常项目，在备注栏填写详细说明
  4. 填写点检人和复检人信息
  5. 选择当前班次（白班或夜班）
  6. 点击"提交点检记录"按钮完成操作

数据管理说明

  1. 导出Excel数据：将点检记录导出为Excel文件
  2. 导入数据：从Excel文件导入点检记录（需特定格式）
  3. 清除所有数据：删除所有点检历史记录（不可恢复）

数据保存说明 Data Storage Note

所有数据保存在浏览器本地存储中，清除浏览器缓存将导致数据丢失。请定期导出数据备份。

All data is stored locally in the browser. Clearing browser cache will delete the data. Please export regularly for backup.
